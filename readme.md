# 🚗 AutoSouq – MERN Car Marketplace

AutoSouq is a full-stack car marketplace web application built using the MERN stack. It allows users to buy, sell, and explore cars through a modern, responsive interface with secure authentication.

## 🔧 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Optional: SQL (for transactions & analytics)
- Authentication: JWT (JSON Web Tokens)

## ✨ Features
- 🔐 User authentication  (JWT)
- 🚘 Create, update, and delete car listings
- 🔍 Advanced search and filtering system
- ❤️ Save favorite cars
- 📱 Fully responsive UI
- 🔒 Secure password hashing and protected routes

## ⚙️ Architecture
- RESTful API design
- MVC backend structure
- Scalable MY SQL database approach

## 🎯 Goal
To provide a secure and user-friendly platform for buying and selling cars online.
